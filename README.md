## Lyrics

Lyrics is a project created with React.js (Hooks), CSS with Bootswatch, Axios to consume APIs.

### APIs

API for lyrics:

- https://lyricsovh.docs.apiary.io/

API for artist info:

- https://theaudiodb.com/

To call two APIs is used **Promise.All**.
